## PIN MAP

| Component    | Pin on ESP32 |
| ------------ | ------------ |
| OLED SDA     | GPIO 21      |
| OLED SCL     | GPIO 22      |
| LED Left     | GPIO 18      |
| LED Right    | GPIO 19      |
| Button Mode  | GPIO 13      |
| Button Reset | GPIO 12      |
| Power        | 3.3V / GND   |
