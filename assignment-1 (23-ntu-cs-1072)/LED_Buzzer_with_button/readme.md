---

## PIN MAP

| Component   | Pin on ESP32 |
|------------|--------------|
| Button     | GPIO 26      |
| LED 1      | GPIO 18      |
| LED 2      | GPIO 19      |
| Buzzer     | GPIO 16      |
| OLED SDA   | GPIO 21      |
| OLED SCL   | GPIO 22      |
| Power      | 3.3V / GND   |

---

## Description

- **Short press** of the button → Toggles LEDs ON/OFF.  
- **Long press** (>1.5s) → Activates the buzzer.  
- OLED displays the current action/message.
